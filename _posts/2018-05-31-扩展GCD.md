### 概念
对于不完全为 0 的非负整数 a，b，必然存在整数对 x，y ，使得 gcd（a，b）=ax+by。

### 代码

    int exgcd (int a,int b,int &x,int &y)
    {
        if (!b) { x=1,y=0; return a; }
        int r=exgcd(b,a%b,y,x);
        y=y-a/b*x;
        return r;
    }
